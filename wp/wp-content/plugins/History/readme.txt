== Description ==
C7 History Developer is a plugin help you with development of your project. you can save a simple history of notes

== Installation ==
1. upload the plugin files to the '/wp-content/plugins/c7-history-developer/' directory, or install the plugin through the WordPress
2. Activate the plugin through the "Plugins" screen in WordPress
3. Go to settings > History Developer
4. Start to save your notes
